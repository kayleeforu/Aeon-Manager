# Privacy Policy

Last updated: June 9, 2026

Aeon Manager is an unofficial desktop app for HoYoLAB check-ins and redemption codes.

## Data Stored Locally

Aeon Manager stores app settings, enabled games, selected regions, local streak data, and local redemption-code history on your device.

If you import HoYoLAB cookies, Aeon Manager stores the required cookie values locally in the app config directory so automation can run.

## Data Sent To HoYoLAB

If automation is enabled, Aeon Manager sends your imported HoYoLAB cookie values to HoYoLAB endpoints to perform check-ins, fetch linked game accounts, and redeem supported codes.

## Data Sent To Aeon Manager Services

If you create an Aeon Manager account, the backend may store your username, recovery UUID, redemption history, and synced streak data.

HoYoLAB cookies are not intentionally sent to the Aeon Manager backend.

## Updates

Aeon Manager may check the configured release endpoint for updates. This may reveal your app version, operating system, and architecture to the release host.

## No Analytics

Aeon Manager does not include analytics or advertising tracking.

## Removing Data

You can remove imported cookies from Settings. You can also delete Aeon Manager's local app config directory to remove local settings and history.
